# Security

Do not post account, browser profile, token, or session details in public
issues.

For security-sensitive reports, use GitHub private vulnerability reporting if it
is enabled for this repository. If it is not enabled, use the security contact
issue template to request a private channel without including secrets, session
data, or reproduction details.

`scinet-queue` stores queue state in the current workspace and browser session
state in a tool-owned browser profile. Treat both as local user data.
